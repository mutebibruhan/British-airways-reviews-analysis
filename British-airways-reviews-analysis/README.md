# ✈️ British Airways Customer Experience Analysis Dashboard

An end-to-end data analytics project analyzing customer review data from British Airways to uncover trends in passenger satisfaction, service performance, aircraft preferences, and customer experience improvement opportunities.

This project combines data cleaning, dashboard development, and business insight generation to transform raw customer feedback into actionable recommendations.

---

## 📌 Project Overview

British Airways receives thousands of customer reviews across multiple countries, aircraft types, and traveler segments. The objective of this project was to analyze customer feedback data and identify:

- Service strengths and weaknesses
- Trends in customer satisfaction over time
- Geographic variations in ratings
- Aircraft performance patterns
- Customer segment insights
- Areas requiring strategic improvement

---

## 🎯 Business Questions

This analysis focused on answering the following key questions:

1. What is British Airways’ average performance across service dimensions?
2. How has service quality evolved over time?
3. Which countries provide the highest and lowest ratings?
4. Which aircraft models receive the strongest customer feedback?
5. Which service dimensions require immediate improvement?
6. How do traveler types and seat classes influence customer satisfaction?

---

## 📊 Dataset Information

- **1,324 verified customer reviews analyzed**
- **Data period:** 2017–2023
- **150+ aircraft models**
- **56 countries represented**
- **Overall average rating:** **4.19**
